# In Developing

以下 crate 正在开发中, 提供内置节点实现和周边工具.

| Crate | 说明 |
|-------|------|
| `arrival-string` | 基于字符串的 Node, 返回固定响应 |
| `arrival-cli-return` | 调用命令行并返回结果的 Node |
| `arrival-serde` | 序列化支持, 可从 TOML/JSON 定义节点 |
| `arrival-toml` | TOML 配置解析入口 |
| `arrival-cli` | 命令行测试工具 |

这些 crate 的 API 可能在后续版本中调整.
